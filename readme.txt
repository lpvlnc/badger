 O projeto foi desenvolvido utilizando a IDE NetBeans 8.2
Abrir o projeto com o netbeans e executar(F6) a classe Game que está no diretório "src/main/Game.java" ou realizar o build (F11), no caso do build será criado dentro do projeto uma pasta chamada "dist" contendo o arquivo Badger.jar que pode ser executado em qualquer ambiente que possua o Java instalado
